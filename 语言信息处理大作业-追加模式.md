# 2018.10.9
**因果关系推出相关性，是在现实世界中普遍认同的观点。假设事物之间有因果关系，那么事物之间必定是存在相关性的。在机器学习和大数据的思维中，人们想通过机器学习和大数据的方法来发现数据之中的某些已经存在的因果关系，及相关性，然而，这些相关性，是人们不能通过简单的推导分析及人类感官而获取到的。

这些事物之间，人类无法获取的因果关系必然会获取人们探索的兴趣，所以在NLP任务中是需要对因果关系进行建模的，将语句中人们可以通过简单的推导分析及人类感官获取到的特征通过一定的形式作为系统的输入**

#2018.10.16
**由因果关系来推测出相关性，这肯定是一个成立的命题，有因果关系的存在相关性，没有丝毫因果关系的事物之间不一定是不存在相关性的，例如a，b事件共同影响着c，a能对c造成后果，b也能对c造成后果，有没有可能存在这样的关系，a与b之间不存在因果关系，又因为a，b都能决定c，所以在考虑c的的情况下，显然a，b是存在相关性的，在把这个域从c拓展到普遍事物上面来，那么a，b之间还是存在也相关性，只是这种相关性变得较弱。所以说到这儿，我觉得这又到了一个数学的问题，我们是怎么个约定法：很接近于0的数是否被认为是0.
近两年来，随着神经网络的再次兴起，NLP的很多任务都可能通过经典的神经网络来开展。就因果关系的建模来说，通过词嵌入的方式，因果关系存在于句法结构中，上下文关系，可以考虑更具有代表性的特征通过级联的方式与词嵌入结合，需要注意的问题可能有，因果关系的数量比例覆盖所有种类：1 vs 1;1 vs n; n vs 1; n vs n

#2018.10.9

**我支持观点一，
拥有更大的数据集，就可以训练更好的语言模型，在配合上不断进步的encode-decode模型，及注意力机制等，机器翻译的效果会更好。 通过句法分析及语义分析等获取被翻译句子中的近乎所有被翻译需要的信息，那么完美翻译句子的能力就是可以被期待的。
人类的翻译也是通过在对语言的理解的，在之前的积累的情况系，在已有的规则之下的，所以人类也不能说自己能完美的翻译任何语言，也会出错，所以机器也一样，只能说是和规则的偏离程度，不可能完完全全的吻合**

#2018.10.16
**首先我认为观点一、二都讲的太绝对了了，不严谨。“完美地”，“再多的数据也不能”，这类词汇我觉得是反科学的，科学是不断进步的，什么东西都不可能是停留在完美的阶段，都可能会进步，或者退步，虽然像随机变量都会服从正太分布一样，都是一个先上升，在下降，整个概率密度等于1的情况，但是什么时候我们能够断定我们是处在最高峰的时候呢，怎么断定我们不是处在某一小区间上面的局部最优的状况呢？现在的神经网络课大数据的理论可以说是以数据为基础的，拥有更多的数据就拥有更多的可能性，有数据就有可能发现数据背后人类不易发现或者尚未发现的信息。所以我相比较下来还是支持观点一
